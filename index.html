Immersive Mode Indicator

This is a custom indicator for Ubuntu Touch to toggle a proof of concept of a new mode for the desktop environment Lomiri wherein some system gestures are disabled to help the user focus on the current app they're using without much distractions. This is specially useful for apps that require different in-app gestures that could conflict with system gestures such as painting/drawing apps and games.

Learn More

Jerk Installer

First of all, this isn't the insult or malicious word. It is in homage to the wonderful software Snap and Crackle 😁 I also love physics! No, not really...... 😂

Jerk is a script for installing custom patches in Ubuntu Touch devices with ease. This is specially useful for patches that only require changes to text-based files such as QML files. It basically copies files from a specifically structured tar file into the system and backs up the original files. Sample patches are changes in the UI in Lomiri and new features in the on-screen keyboard.

Ambot Installer is an Ubuntu Touch app that provides a GUI for installing prebuilt Jerk packages. It is available on the OpenStore as a click package.
